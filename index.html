Seismic Noise Dumping Simulation

A web tool to model seismic noise filters used in ground-based laser interferometers. Users can adjust physical parameters and noise components to simulate how a multistage pendulum can be used to dampen seismic disturbances. The system uses a very simple double pendulum with equal masses and equal lengths. For any questions, email me at chinchiofilippo@gmail.com.
